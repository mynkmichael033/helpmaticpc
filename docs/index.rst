PC Matic User Guide | Installation, Scans & Troubleshooting
===========================================================

Overview
--------

PC Matic is a security and performance software designed to protect computers from malware, ransomware, and other online threats. This guide provides step-by-step instructions for installing, using, and troubleshooting PC Matic on Windows and macOS systems.

System Requirements
-------------------

Before installing PC Matic, ensure your system meets the minimum requirements:

- **Operating System:** Windows 10 or later, macOS 10.14 or later  
- **RAM:** 2 GB minimum (4 GB recommended)  
- **Disk Space:** At least 500 MB free  
- **Internet Connection:** Required for updates and cloud-based scans  

Installation
------------

Follow these steps to install PC Matic on your computer:

1. Download the PC Matic installer from the official website.  
2. Locate the downloaded file and double-click it to start installation.  
3. Follow the on-screen instructions to complete installation.  
4. When prompted, enter your license key to activate the software.  
5. Complete the setup wizard to enable automatic updates and scanning.  

*Troubleshooting Installation Issues:*

- Close any other antivirus software before installation.  
- Restart your computer if the installation fails.  
- Verify that your system meets the minimum requirements.  

Running Scans
-------------

PC Matic offers different types of scans:

- **Quick Scan:** Checks critical system files and running processes.  
- **Full Scan:** Scans the entire computer for malware, ransomware, and other threats.  
- **Custom Scan:** Allows selection of specific files or folders to scan.  

*How to Run a Scan:*

1. Open the PC Matic application.  
2. Select the type of scan you want to perform.  
3. Click **Start Scan**.  
4. Review the results and follow the recommendations for any detected threats.  

Updating PC Matic
-----------------

Keeping PC Matic up to date ensures maximum protection:

1. Open the PC Matic application.  
2. Navigate to **Settings → Updates**.  
3. Click **Check for Updates**.  
4. Install any available updates when prompted.  

Common Troubleshooting
----------------------

**Problem:** Software does not launch  
- Ensure your operating system is up to date.  
- Restart the computer.  
- Reinstall PC Matic if necessary.  

**Problem:** Scans do not complete  
- Close other running applications.  
- Make sure the application has permission to access all drives.  
- Restart the computer and try again.  

**Problem:** System performance slows after installation  
- Check scan schedules to avoid running scans during high-use hours.  
- Exclude large folders that do not need frequent scanning.  

Best Practices
--------------

- Keep PC Matic updated at all times.  
- Schedule regular system scans.  
- Avoid downloading software from untrusted sources.  
- Backup important files regularly.  

Conclusion
----------

PC Matic provides a comprehensive solution for securing and optimizing computers. By following this guide, users can install, configure, and maintain PC Matic effectively, ensuring a safe and smooth computing experience.
